# PlayLog
Log your plays.

## Project Structure

```
PlayLog/
├── frontend/   # React + Vite + TypeScript
└── backend/    # Python + FastAPI (coming soon)
```

## Frontend

### Requirements
- Node.js 18+
- npm

### Setup

```bash
cd frontend
npm install
```

### Running

```bash
# Development server
npm run dev

# Production build
npm run build

# Preview production build
npm run preview
```
